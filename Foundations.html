Data Dunkers Foundations

This introductory format focuses on engaging students with the core concepts of data science through the lens of basketball. Utilizing both their own data and data from the NBA/WNBA, participants will learn to make performance improvement recommendations for basketball jump shots.

Key Components

  • Basic data analysis and storytelling
  • Learn to use Jupyter Notebook for data input processing and visualization
  • Develop and pitch recommendations based on data analysis

Driving Question for Data Dunkers Fundamentals

"What can the data from professional basketball players' performance, like Pascal Siakam's jump shots, tell us about the principles of good shooting and how can we apply these insights to improve our own game?"

Learning Modules

These modules represent the core of the Data Dunkers program. Using data collected by the students they learn about the basic process that data scientists go through. They then apply these skills to the larger data set of the NBA or WNBA to identify a trend or specific question. They will develop a recommendation for their personal or class performance based on this larger trend and present that in a pitch.

Introduction to the Program

Purpose: The introductory module of the Data Dunkers program aims to inspire curiosity among students about the intersection of basketball and data science. We'll establish a shared understanding of the program’s objectives and how our activities will explore the power of data analysis in sports. By sparking questions and engaging discussions from the start, we ensure students are not only informed but also excited to discover what data can reveal about basketball performance and beyond.

Learning Focus

Concept: Integrated Understanding of Data Science in Basketball

  • Outcome: "I can explain the goals of the Data Dunkers program and understand how data analysis can improve basketball performance using Pascal Siakam's statistics as an example."

  • ISTE Standards: Empowered Learner (1d) - "I can effectively use technology and digital tools to enhance my learning." Knowledge Constructor (3c) - "I can gather and make sense of information from digital sources to build knowledge."

Concept: Foundation for Inquiry and Collaborative Learning

  • Outcome: "I can ask meaningful questions about basketball data and work with my peers to share our goals and expectations for the program, establishing a supportive learning community."

  • Aligned ISTE Standards: Innovative Designer (4a) - "I can use a design process to solve problems by asking questions and generating ideas." Global Collaborator (7b) - "I can collaborate with others using digital tools to explore data science and basketball."

  • Review of 'The Data Dunkers' Code' to establish program norms and expectations. [Proposed]

Tools of Data Science

Purpose: This module equips students with the knowledge to navigate and utilize Jupyter Notebooks for basketball data analysis, integrating Python programming for data filtering and visualization. By combining these tools and techniques, students embark on an initial exploration of data science, learning to ask critical questions and interpret data through hands-on activities. The module aims to foster a foundational understanding of how technology can unravel insights into basketball performance, encouraging curiosity and analytical thinking.

Learning Focus

Concept: Utilizing Jupyter Notebooks for Data Science with Python

  • Outcome: Students will understand how to navigate Jupyter Notebooks, distinguishing between markdown and code cells, and execute Python code for basic data analysis tasks. This includes filtering basketball data and creating visualizations, thus laying the foundation for both data science exploration and programming skills.

  • ISTE Student Standards: Computational Thinker (5a) - "I can use technology to identify and solve problems by creating new useful or imaginative solutions." Innovative Designer (4d) - "I can show evidence of learning through creating solutions that involve programming languages and tools."

Concept: Data Exploration

  • Outcome: Students will engage in data exploration by observing and interpreting basketball data through filtering and visualization. They will learn to ask informed questions and reflect on the insights gained from data analysis within Jupyter Notebooks.

  • ISTE Student Standards: Knowledge Constructor (3c) - "I can curate information from digital resources to build knowledge and draw conclusions." Computational Thinker (5d) - "I understand how to break down problems into smaller parts, recognize patterns, and create algorithms to analyze data and draw conclusions."

Launch

"We are going to actually begin to analyze some data now. This activity will walk you through our primary tool: The Jupyter Notebook and how we use this with data science. Remember as Data Scientists, this activity is meant to have you explore and learn. If you are new to coding, you will be learning how to edit code to get the revised results you want. Work with one another, ask questions, look-up answers, and ask for help if you are stuck."

Activities

  • Mini-Basketball Labyrinth: Engage students with a hands-on task in a Jupyter Notebook to learn about cells and basic Python code as applied to data science and basketball. Students will open the Notebook in Callysto and then be able to walk through the prompts and learn about the basic tools we will be using.

  • Facilitate a Reflection at the end: Data Science looks at the information it has and asks “What do I notice?”

    • What do you notice about the notebooks themselves and how they work?
    • What do you notice about the data that you looked at?
    • Different columns and how we filtered it
    • The ways we can visualize it
  • Add the badge to their Google slide page about themselves if you created those.

Materials

  • Mini-Basketball Labyrinth Jupyter Notebook: Prepared with instructional content and basketball data for the Mini-BB Labyrinth activity.
  • Laptops/Chromebooks: Computers cannot be “end of life.” They will need to be tested for ability to access and run the Jupyter Notebooks.
  • Class Google Slide Deck from last Module.
  • Jupyter Notebook Demo: So that you can review of go further in depth with the students.

Input Stage: Our Data

Purpose: This module guides students through the essential steps of collecting, inputting, and beginning to analyze their basketball shot data. By engaging in both theoretical discussion and practical application, students will gain firsthand experience in the data collection process and understand its importance in drawing meaningful conclusions. The activities are designed to highlight the value of systematic data collection and the power of exploratory data analysis in identifying patterns and insights that can enhance basketball performance.

Learning Focus

Concept: Data Collection, Input, and Organization

  • Outcome: "I can identify different types of data from basketball shots, use digital tools to collect this data during practice, and input it into a database or spreadsheet for comparison and analysis."

  • ISTE Student Standards: Computational Thinker (5b) - "I can collect and analyze data to identify solutions and make informed decisions." Innovative Designer (4b) - "I can use a variety of technologies to identify and solve problems by creating new useful or imaginative solutions."

Concept: Introduction to Exploratory Data Analysis

  • Outcome: "I can use basic statistical concepts to begin analyzing my basketball shot data, understanding the importance of large datasets for identifying patterns that can improve my performance."

  • ISTE Student Standards: Knowledge Constructor (3d) - "I build knowledge by actively exploring real-world issues and problems, developing ideas and theories, and pursuing answers and solutions." Computational Thinker (5d) - "I can decompose problems into smaller parts, use data to identify patterns, and design algorithms to achieve outcomes."

Launch

Video of Students Shooting Baskets (OR just some from NBA/WNBA)

Activities

  • Data Types Discussion: After re-watching a segment of the basketball shooting video, brainstorm and list the various data points (e.g., shot distance, success/fail, shot angle) that can be collected for each shot attempt.
    Optional extension: Collaboratively create a form with those data points OR review the data collection tool.

  • Data Collection Workshop: Guide students through either creating a digital form or utilize our pre-made tool for recording their shot data during the practice session.

  • Go Shoot some hoops: Students should either go down to a real basketball court or use an in room "hoop" with crumbled paper. They should collect data on their shots for you to use.

  • Data Input Tutorial: Upon returning, demonstrate how to input the collected data into a simple database or spreadsheet, including historical data for a richer data set.
    Important Discussion: Why historical data? Need larger data set to identify patterns. Show video of small numbers and how they can be shifted to say something that isn’t accurate. Full video is good but really start with the 1:30 mark.

  • Exploratory Data Analysis: Introduce basic statistical concepts to analyze the shot data, emphasizing the importance of large datasets for identifying meaningful patterns.

Materials List

  • Shooting Video: Footage of students or professional players shooting baskets.
  • Data Collection Tool: Digital forms or pre-made tool (Form | Results Sheet ) for students to record their shot data.
  • Video on Data Misinterpretation: To discuss the importance of historical data and large data sets, highlighting how small sample sizes can lead to inaccurate conclusions.
  • Teacher Guide for Input Stage.

Input Stage: NBA and WNBA Data

Purpose: The purpose of this module is to demonstrate the value of analyzing large comprehensive datasets such as those from the NBA/WNBA in gaining deeper insights into basketball performance. It aims to cultivate students' ability to use data analysis for enhancing understanding and developing strategies for personal and team performance improvement.

Learning Focus

Concept: Importing and Analyzing NBA/WNBA Data

  • Outcome: "I can understand the process of importing large datasets and engage in initial analysis to ask informed questions about basketball performance."

  • ISTE Student Standards: Computational Thinker (5b) - "I can collect data or identify relevant data sets, use digital tools to analyze them, and represent data in various ways to facilitate problem-solving and decision-making." Knowledge Constructor (3c) - "I can curate information from digital resources to create collections of artifacts that demonstrate meaningful connections or conclusions."

Launch

Highlight the significance of using comprehensive datasets like the NBA/WNBA's to analyze shot accuracy and its implications on player performance improvement and what recommendations we could develop for our own performances.

Activities

  • Import: Through a screen share, the teacher demonstrates the process of importing the larger dataset, discussing data types and sources. Ask questions to reinforce the process that they had done earlier.

  • Initial Observations: Engage students with initial data analysis tasks focusing on simple statistical measures (e.g., average shot distance, shooting accuracy). Prompt discussion on observations:

    • What do you notice about the data?
    • What are the other columns we have?
    • What other questions could we ask of the data if we used some of this other data?

Materials

  • Screen Sharing Tool: For the teacher to share the data importation process and initial data analysis with students.
Process Stage: Previewing and Preparing Our Data

Purpose: This module aims to develop students' foundational skills in data manipulation using Jupyter Notebooks and the pandas library. By focusing on essential techniques such as column manipulation, data filtering, and the creation of calculated columns, students will learn to transform raw data into actionable insights about basketball shooting performance.

Learning Focus

Concept: Data Manipulation Skills in Jupyter Notebooks

I Can Statement: "I can use Jupyter Notebooks and the pandas library to manipulate basketball shot data by renaming columns, filtering data, sorting data, and creating calculated columns to uncover insights about shooting performance."

ISTE Student Standards: Computational Thinker (5c) - "I can break down data into component parts, perform operations on it, and analyze the results to make conclusions." Innovative Designer (4c) - "I can use a variety of technologies to create new, useful or imaginative solutions."

Launch

Begin with a quick walkthrough of the dataset collected during the input stage. Highlight the significance of each column and the potential insights they can provide about basketball shots.

Activities

Using Different Questions that you could solve with the data, first demonstrate and then have students:

  • Rename columns
  • Filter data
  • Sort data
  • Create calculated column

Materials

  • Basketball Shot Dataset: Collected in earlier stages ready for manipulation.
  • Jupyter Notebook: Pre-set for data manipulation exercises.
  • Instructional Guide: Step-by-step instructions for renaming columns, filtering, sorting, and creating calculated columns in pandas.
  • Data Analysis Questions: A list of questions to guide students' exploration of the dataset, such as the average shooting accuracy from different distances, how shooting accuracy varies between practice sessions, and more.

>Filtering Data: NBA/WNBA

Purpose: This module aims to apply the foundational data science skills students have developed to a larger real-world dataset focusing on professional basketball performance from the NBA/WNBA. It is designed to deepen students' understanding of data analysis by exploring how different variables influence basketball outcomes, embodying the essence of data science: extrapolating from specific observations to broader insights.

Learning Focus

Concept: Real-World Data Analysis Application

  • Outcome: "I can apply data manipulation techniques using pandas in Jupyter Notebooks to explore professional basketball data, testing hypotheses and uncovering insights related to player performance and game strategies."

  • ISTE Student Standards: Computational Thinker (5c) - "I can use digital tools to break down data into component parts and analyze patterns to study the impact of different variables on basketball performance." Knowledge Constructor (3d) - "I can critically curate a variety of resources using digital tools to construct knowledge, produce creative artifacts, and make meaningful learning experiences for myself and others."

Launch

Remind students of the questions we asked about the data set. Walk them through opening their group’s Notebook and make sure they know the question they are asking. Remind them that this is the core of data science. Taking a smaller observation and then asking how other variables affect the results.

Activities

  • Group Exploration: Divide students into small groups, assigning each a specific question or hypothesis about the NBA/WNBA data (e.g., "Does the side of the court that the shot was taken on further impact the shooting accuracy at different distances?").

  • Data Manipulation Challenge: Facilitate a hands-on session where students use pandas to filter, sort, and analyze their assigned dataset segment. Encourage them to note any interesting patterns or anomalies. Have the students use the structured Notebook [ERIC’S] to help them filter and select data relevant to their question.

  • Reflection and ‘ahah’s’: Have students add to their slide(s) what they were focused on in terms of a question and what they noticed.

Materials

  • Group Notebooks: Pre-prepared Jupyter Notebooks each tailored to a specific question or hypothesis about the NBA/WNBA data.
  • Instructional Guide: Guide for teacher to support students in asking their questions.
  • Reflection Slide Template: A template for students to document their assigned question, their analytical process, and any significant findings or 'aha' moments int eh Class Slide Deck.
Output Stage: Examining and Visualizing Our Data

Purpose: This module is designed to enhance students' ability to communicate data analysis findings by teaching them how to select and create effective visualizations. Utilizing pandas and Plotly within Jupyter Notebooks, students will learn to craft compelling stories with their data through the use of various plot types such as bar graphs, scatter plots, pie charts, and histograms. The focus is on developing a nuanced understanding of how visual representations can alter perceptions of data and underscore certain narratives or findings.

Learning Focus

Concept: Data Visualization Techniques

  • Outcome: "I can select the appropriate type of data visualization such as bar graphs, scatter plots, pie charts, or histograms using pandas and Plotly in Jupyter Notebooks and customize these visualizations to effectively communicate my data analysis findings."

  • ISTE Student Standards: Knowledge Constructor (3c) - "I can critically curate a variety of resources using digital tools to construct knowledge, produce creative artifacts, and make meaningful learning experiences for myself and others." Creative Communicator (6a) - "I can communicate clearly and express myself creatively for a variety of purposes using the platforms, tools, styles, formats, and digital media appropriate to my goals."

Launch

Begin the module by displaying a dataset to the students and engage them with the question "What story does this data tell us?" Proceed by showing a graph or visualization derived from the same dataset, prompting further discussion with "How does this visualization change our understanding or highlight certain aspects of the data?" Finish by asking "What decisions did I need to make to produce the visualization?" and explain that as data scientists, the creation of visualizations is a methodical process aimed at telling the story hidden within the data, showcasing discoveries and insights.

Activities

Conduct a hands-on session where the teacher demonstrates the step-by-step process of creating a bar graph using Plotly with the students’ own data set (plus historical) in the Notebook. Students will walk through the same process in their Notebook. The process will include selecting appropriate data, deciding on the type of graph that best represents the data, and any customization of the visualization to make it more informative. Students will be provided with time to create their own visualization. They will then explain what they wanted their visual to 'say' and how they designed their graph to say it. Possible questions they could ask of the data to visualize include:

  • Accuracy vs. Distance Exploration: "How does our shooting get better or worse as we move farther from or closer to the net, and is there a sweet spot where we're hitting the most shots?"
  • Pattern in Shot Sequence: "Do we tend to make more shots at the beginning, middle, or end of our shooting practice, and what does that say about how tired we get or how we warm up?"
  • Improvement Over Time: "Looking at how we did over several sessions of shots, are we getting better? Can we spot any trends or unexpected changes in how many shots we're making?"

Materials

  • Students' Dataset: Previously collected and historical data for use in visualization exercises.
  • Jupyter Notebooks: For students to walkt through the process.
  • Visualization Guide: A step-by-step tutorial on creating different types of visualizations with Plotly, including considerations for selecting data and graph types.

Making Recommendations

Purpose: This module builds upon students' ability to analyze and visualize data by guiding them to the next logical step: making informed recommendations based on their findings from the NBA/WNBA dataset. It's designed to foster a deeper engagement with the data, encouraging students to transition from observers to active participants in the discourse on basketball performance improvement.

Learning Focus

Formulating Actionable Recommendations

Outcome: "I can use the insights gained from analyzing the NBA/WNBA dataset to formulate specific actionable recommendations for improving basketball performance, demonstrating the application of data-driven decision-making."

ISTE Student Standards: Knowledge Constructor (3d) - "I build knowledge by actively exploring real-world issues and problems, developing ideas and theories, and pursuing answers and solutions."

Communicating Recommendations Effectively

Outcome: "I can clearly and compellingly communicate my recommendations, supported by data visualizations and analysis, to convey how these suggestions can lead to improvements in basketball performance."

ISTE Student Standards: Creative Communicator (6c) - "I can communicate complex ideas clearly and effectively by creating or using a variety of digital objects such as visualizations, models, or simulations."

Launch

Start this section by discussing the significance of transforming data analysis into actionable insights. Introduce the concept of an "elevator pitch" as a concise and persuasive way to present recommendations, highlighting its importance not only in sports analytics but across various industries. Showcase an example of an effective pitch, emphasizing clarity, brevity, and the power of persuasion to illustrate how students can aim to communicate their findings and recommendations.

Activities

Guide students through the process of summarizing their key findings from the data analysis and visualizations they've worked on. This summary should focus on the insights gleaned from both their own data and the NBA/WNBA dataset. Encourage students to identify patterns or standout observations that could lead to actionable recommendations for improving basketball performance.

  • Summary of Key Findings: Each student pair should start by drafting a concise summary of their most significant findings. Encourage them to focus on clear and compelling evidence that supports their insights.
  • Formulating Recommendations: Based on these findings, students will then develop one to two core recommendations aimed at enhancing shot performance or other aspects of basketball play. Encourage them to think creatively about training methods, strategies, or techniques that could be derived from their analysis.
  • Crafting an Elevator Pitch: Instruct students on how to distill their findings and recommendations into a succinct elevator pitch using the slide deck with the visuals they have already captured. This pitch should be no longer than 3 minutes and compellingly presenting their analysis and suggesting practical applications. If time allows, have student pairs practice their pitch with another group and receive feedback.

Materials

  • Jupyter Notebooks: Previously analyzed dataset for drawing insights and formulating recommendations.
  • Examples of successful or good 'elevator pitches': These can be searched for and shared with the students.
  • Recommendation Report Template: A structured template for students to draft their recommendations, ensuring they articulate their rationale and supporting evidence effectively.
  • Presentation Tools: Access to digital presentation tools or platforms for students to share their recommendations, possibly including features beyond the slide deck for more dynamic presentations.
  • Feedback Mechanism: Tools or protocols for providing feedback on students' recommendations, encouraging a reflective and iterative approach to data-driven decision-making.
Celebration

Purpose: The in-class celebration aims to acknowledge and showcase the analytical achievements of students in the Data Dunkers program. It serves as an opportunity for students to share their NBA/WNBA data analysis projects, reflect on their learning journey, and receive recognition for their hard work and insights. This celebration marks the culmination of their efforts, bridging their detailed analytical work with the broader community and preparing them for future endeavors in data science.

Activity: It is recommended that you engage with an event that celebrates the students' work through a sharing of their recommendations to an appropriate audience. This could be to the coach and basketball team, family, or other classes. If a larger celebration is not possible or before your larger celebration event, we suggest you still host an in-room celebration to honor the students' dedication and achievements. This moment will serve as an acknowledgment of their efforts, providing an opportunity for instructors to offer personalized commendations and for students to reflect on their growth and learning experiences. This celebration will act as a bridge connecting the rigorous analytical work completed by the students with the anticipation of sharing their accomplishments with a broader audience. This can be done quickly if time is a factor but be sure to:

  • Recognize their Achievements: Highlight individual and group contributions, showcasing the diverse range of insights and recommendations developed through the program. This recognition not only validates the students' hard work but also boosts their confidence as they prepare for the public presentation of their findings.
  • Reflect on the Journey: Facilitate a reflective discussion that encourages students to share their learning experiences, challenges overcome, and personal growth moments. This reflection will deepen their understanding of the program's impact on their analytical and personal development.

Possible Celebration Formats

  • Presentation Lightning Pitch Session: Students share their pitch summarizing their project, key insights, and recommendations. This fast-paced format encourages clarity and excitement, allowing each group or individual a spotlight moment. The session can conclude with a Q&A from the class, fostering interaction and deeper engagement.
  • Presentation Carousel: Set up stations around the classroom, each dedicated to a student or group project. Students rotate between stations at set intervals, allowing for more intimate discussions and detailed exploration of the projects. This format encourages personalized feedback and in-depth conversations about the students' work.
  • Digital Presentation Gallery: Students create short presentation videos summarizing their projects, key insights, and recommendations. These videos are then uploaded to a digital platform accessible to the class, such as a class website or a shared online folder. Students and guests can view the presentations at their own pace, and the platform allows for asynchronous feedback, questions, and reflections through comments or a discussion forum.
  • Peer Review Circles: Divide the class into small groups or circles where each student or team has the opportunity to present their findings and recommendations briefly to their peers. After each presentation, the group engages in a structured feedback session offering constructive comments, asking clarifying questions, and discussing the insights presented.

Materials

  • Certificates of Achievement: Personalized certificates recognizing each student's participation and contributions.
  • Project Display Materials: Tools for showcasing student work, such as posters, digital slides, or printed reports.
  • Reflection Activity Supplies: Note cards or digital platforms for students to write or share their reflections on the journey.
  • Award Ribbons or Stickers: Small tokens to acknowledge specific achievements, such as "Best Data Visualization" or "Most Innovative Recommendation."
